# Instagram Fast Forward Chrome Extension

Hold spacebar to fast forward Instagram videos at 2x speed, just like the fast-forward feature on the Instagram mobile app.

## Install

### Option 1: Chrome Web Store

1. Go to the [Chrome Web Store](https://chromewebstore.google.com/)
2. Search for **"Instagram Fast Forward"**
3. Click **Add to Chrome**

### Option 2: Load Unpacked (Manual)

1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions`
3. Enable **Developer mode**
4. Click **Load unpacked** and select this folder
5. Go to instagram.com and hold spacebar on any video
